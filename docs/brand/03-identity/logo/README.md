## 2. Direção criativa

### 2.1. Referências canônicas (estudar antes de desenhar)

| Referência | O que extrair |
|-----------|--------------|
| [Kibana](../../references/inspirations/01-brands/01-kibana.md) | Letterform com **gesto orgânico/fluido**; acento dourado em detalhe pequeno (substituindo um diacrítico) |
| [Studio88 R](../../references/inspirations/01-brands/02-studio88-r-monogram.md) | Possibilidade de versão monoline; ouro foil; linha "extra" que escapa do letterform |
| [Peppermint](../../references/inspirations/01-brands/03-peppermint.md) | **Anti-padrão:** não usar elementos botânicos (folha, gota); micro-elemento orgânico **abstrato**, não literal |

### 2.2. Princípios para o logo KEYRA

1. **Letterform-based.** O logo nasce da tipografia, não de um símbolo abstrato + nome. KEYRA é palavra; o logo é a palavra elevada.
2. **Gesto orgânico discreto.** Em algum ponto do letterform — provavelmente no "K" inicial ou no "Y" — uma curva orgânica/fluida que escapa da geometria rígida. **Movimento sutil, não estilismo gritante.**
3. **Detalhe dourado em diacrítico.** O "i" da KEYRA pode carregar o ponto dourado (substituindo o pingo). Ou um glifo dourado discreto entre as letras. **Um elemento gold, não múltiplos.**
4. **Família serif editorial.** Custom, mas modelado em **Fraunces 600/700** (família já adotada). Não usa Fraunces direto — refina-se a partir dela.
5. **Funciona em 16px.** Monograma "K" precisa ler em favicon. Logotipo precisa ler em assinatura de email a 24px de altura.
6. **Funciona em monocromático.** Desenho sólido em uma cor (cocoa-900 ou ivory-50) — sem o ouro — ainda tem que ser KEYRA. Ouro é elevação, não dependência.
7. **Não usa ícone botânico.** Sem folha, sem gota, sem flor. KEYRA não é spa, é **software**.

### 2.3. O que evitar

| Anti-padrão | Por quê |
|-------------|---------|
| Símbolo + nome separados (logo Apple, Nike) | Quebra o conceito letterform-based; KEYRA é palavra primeiro |
| Tipografia geométrica rígida (Futura, Helvetica) | Não bate com editorial luxury beauty |
| Elementos botânicos (folha, planta, gota d'água) | Sinaliza spa/skincare; KEYRA é categoria SaaS |
| Sparkle/star/glitter | Vulgar, geração errada |
| Neon/glow/animação 3D | Brega, anti-quiet luxury |
| Cores múltiplas no mark | Logo é mono ou bicolor (cocoa + gold) |
| Initialism KYRA, KRA, K. | KEYRA é a palavra completa — assinatura é a palavra |

## 3. Avaliação 5-pillar (Sagi Haviv)

Quando 3-5 conceitos estiverem desenhados, avaliar contra:

| Pilar | Pergunta-teste |
|-------|---------------|
| **1. Appropriate** | É consistente com o que a KEYRA faz e para quem fala? |
| **2. Distinctive** | Diferencia visualmente de Trinks, Belle, Gestek, Kamino, Conta Azul? |
| **3. Simple** | Funciona em 16x16px? Funciona em monocromático? |
| **4. Memorable** | Após 3 segundos vendo, alguém consegue rascunhar? |
| **5. Versatile** | Aplica em ivory-50, cocoa-900, sobre foto, em foil dourado, em redação SVG inline? |

Logo só vai pra produção quando os 5 forem **sim**.

## 4. Especificações técnicas

### 4.1. Cores aprovadas

| Versão | Cor primária | Cor de detalhe |
|--------|-------------|----------------|
| Padrão | `cocoa-900` `#2B1810` | `gold-500` `#B8923A` (pingo do "i" ou similar) |
| Sobre escuro | `champagne-200` `#F0D8A8` | `gold-500` `#B8923A` |
| Mono claro | `cocoa-900` `#2B1810` | — |
| Mono escuro | `ivory-50` `#FAF6EE` | — |
| Premium foil | `gold-500` `#B8923A` | — |

### 4.2. Tipografia base

A custom letterform parte de **Fraunces 600** (Google Fonts) com modificações nos seguintes glifos:
- "K" — gesto orgânico no traço diagonal
- "i" — pingo substituído por marca dourada (badge/glifo discreto)
- "A" — possível abertura editorial (apex)

Decisões finais com Sagi.

### 4.3. Clear space

Espaço respiratório mínimo ao redor do logo = **altura da letra "K"** em qualquer direção. Em peças premium (capas, hero), usar **2× altura do K**.

### 4.4. Tamanho mínimo

| Suporte | Largura mínima |
|---------|----------------|
| Digital (web, app) | 80px de largura para logotipo; 16px para monograma |
| Impresso | 25mm de largura para logotipo; 8mm para monograma |
| Foil/embossed | 40mm de largura mínimo (para garantir registro do foil) |

### 4.5. Posicionamento canônico

- **Header de produto:** canto superior esquerdo, monograma + wordmark, 32px de altura
- **Capa de proposta:** centralizado superior, logotipo vertical, ~120px de altura
- **Header de email:** canto superior esquerdo, logotipo horizontal, 24px de altura
- **Avatar social:** monograma centralizado em fundo `cocoa-900` ou `ivory-50`
- **Marca d'água em peça gráfica:** `gold-500` em opacity 18%, canto inferior direito, ~10% da largura da peça
