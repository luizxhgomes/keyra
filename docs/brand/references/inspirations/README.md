## DNA destilado (leitura obrigatória)

[`02-cultural-direction/editorial-beauty-luxury.md`](02-cultural-direction/editorial-beauty-luxury.md) contém:

- **6 princípios visuais não-negociáveis** (calor sempre, tipografia é estrela, ouro precioso, mulher real, tátil, drama sem ostentação)
- **Tribo cultural correlata** (Glossier, Aesop, Hermès, Le Labo, Vogue Brasil, Quiet Luxury)
- **Anti-direções** (tech-bro clean, spa fofo, ERP corporativo, beauty hyperfeminino, luxo ostentação)
- **Calibragem brand vs product** — como a expressão completa se destila em UI funcional

## Como adicionar mais inspirações

Quando quiser adicionar nova referência:

1. Salvar imagem em `_assets/` com nome descritivo
2. Criar arquivo em `01-brands/` (se for marca específica) ou `02-cultural-direction/` (se for movimento/estética geral)
3. Seguir o template usado em `01-kibana.md`:
   - Imagem
   - O que é
   - Por que está aqui
   - O que aproveitar para KEYRA (com aplicação concreta)
   - O que **não** copiar
   - Decisões que isso alimenta

## Anti-referências (a coletar quando relevante)

Pasta opcional `03-anti-references/` (a criar quando houver) — para documentar marcas que **NÃO** queremos parecer. Útil quando o time tiver dúvida sobre direção.

Os 6 anti-padrões já documentados em [`02-cultural-direction/editorial-beauty-luxury.md`](02-cultural-direction/editorial-beauty-luxury.md) cobrem o macro:

- Spa/wellness fofo

- ERP corporativo
- Skincare clean millennial
- Luxo ostentação
