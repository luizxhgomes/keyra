## 10 Takeaways para Não Esquecer

1. **Belle é bootstrap puro 17 anos, cash-flow positivo, zero VC, capital social R$ 170k.** Não tem munição para guerra de preço.
2. **"Parceria TOTVS" é ilusão** — Belle é só cliente do ATS TOTVS RH. Desmistificar no conteúdo.
3. **ARR estimado R$ 4,8-11,3M** (2.700 clínicas × R$ 150-350 declarativo; R$ 400-800 real com add-ons).
4. **Ticket real é 3-4× o anunciado** — base R$ 170 vira R$ 619-769/mês com WhatsApp + BI + Marketing + NFS-e. Bait-and-switch perceptível.
5. **Fidelidade 1-2 anos + multa 50% é lock-in contratual, não por valor.** Clientes ficam por custo de saída. KEYRA com zero trava vence.
6. **WhatsApp via Speedchat/BelleMessage baniu números de clientes** — dor judicializável e RA público. Cloud API Meta oficial é seguro + feature-âncora KEYRA.
7. **Bug recorrente em controle de pacote** ("12 sessões em vez de 10") — forçar dona a manter planilha paralela. Trilha de auditoria imutável = headline KEYRA.
8. **IA da Belle é bolt-on** (BelleChat + Gestor IA são chamadas LLM externas sobre monolito legado). KEYRA AI-native por design é moat defensável 18-24m.
9. **CEO Rafael Thibes é autoridade de palco (Conference 23k) mas invisível digital** (LinkedIn tímido, sem podcast, sem YouTube). Founder-led moderno do Luiz vence em feed.
10. **Janela competitiva: 18-24 meses** antes que Nova UX (ainda em lançamento 2025) + reação AI-native fechem. O relógio está rodando.
