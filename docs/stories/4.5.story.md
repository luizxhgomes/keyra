# Story 4.5: Agenda do Dia no Dashboard

## Status

Ready

## Story

**Como** owner,
**Eu quero** ver os agendamentos de hoje direto no dashboard,
**Para que** eu saiba o que está agendado sem precisar abrir a `/agenda` (FR-DA-06).

## Complexidade

**T-shirt:** S (~2 pontos · 0,3 dia)

## Contexto

Reusa `v_dashboard_kpis.appointments_today` para o contador. Para a lista, faz query simples em `appointments` filtrando por `starts_at` no dia corrente. Componente embutido no `/dashboard` (Story 4.4).

## Acceptance Criteria

### AC1 — Card "Agenda de hoje" no dashboard

1. Listagem de até 10 agendamentos do dia corrente, ordenados por `starts_at` ASC.
2. Cada linha: horário (HH:mm), paciente, serviço, profissional, status (badge).
3. Click em uma linha → abre `/agenda` na visão do dia (mantém scroll do dashboard via query string).

### AC2 — Resumo

1. Header do card: "X agendamentos hoje · Y concluídos · Z cancelados".
2. Empty state: "Sem agendamentos para hoje. Use a [agenda](/agenda) para criar."

### AC3 — Atualização

1. Server Component; revalida via `revalidatePath('/dashboard')` quando agenda muda (Stories 2.5/2.6 já chamam).

## Riscos

2. Click navega para `/agenda` no dia.
3. typecheck + lint + build verdes.
