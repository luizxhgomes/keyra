# Story i6.2: Cenários what-if

## Status

Draft

## Story

**Como** Camila pensando em estratégias de crescimento,
**Eu quero** simular "+10% upsell em todos os atendimentos" ou "+5 atendimentos/semana",
**Para que** eu visualize impacto financeiro antes de investir tempo/dinheiro em uma decisão estratégica.

## Complexidade

**T-shirt:** M (~5 pontos · 1 dia)

## Acceptance Criteria

### AC1 — Página `/inteligencia/cenarios`

1. Form com sliders independentes:
   - Δ ticket médio (-30% a +30%)
   - Δ volume (-50% a +50%)
   - Δ no-show (-10% a +10%)
   - Δ custo variável (-20% a +20%)
   - Δ custo fixo (-20% a +20%)
2. Output em tempo real (debounced 300ms): Receita / Custos / Lucro projetados para próximos 30/90/180 dias.

### AC2 — Cenários pré-configurados

1. Botão "+10% ticket via upsell".
2. Botão "+5 atendimentos/semana".
3. Botão "Reduzir custo insumo 15%".
4. Botão "Aumentar margem 5pp".

### AC3 — Salvar e comparar

1. Botão "Salvar este cenário" → grava em `whatif_scenarios (id, org_id, name, deltas jsonb, created_at)`.
2. Comparação 2-up: 2 cenários lado-a-lado com KPIs.

### AC4 — Cálculo

```
Receita = base_receita × (1 + Δticket) × (1 + Δvolume) × (1 - Δnoshow)
Custo Var = base_custo_var × (1 + Δcusto_var) × (1 + Δvolume)
Custo Fixo = base_custo_fixo × (1 + Δcusto_fixo)
Lucro = Receita - Custo Var - Custo Fixo
```

Conforme `squads/squad-keyra-intelligence/config/formulas-projecao.md`.
