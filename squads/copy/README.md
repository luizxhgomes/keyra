# Copy Squad

Squad especialista em copywriting de resposta direta. 12 agents lendarios em 4 tiers.

## Ativacao

```bash
/copy                    # Ativa o Copy Chief
@copy:copy-chief         # Direto no orquestrador
@copy:eugene-schwartz    # Agent especifico
```

## Pipeline

```
Pedido → Copy Chief (Tier 0 diagnosis)
```
