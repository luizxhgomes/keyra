# Copy Quality Checklist

General quality checklist for all copy deliverables.

## Pre-Flight

- [ ] Tier 0 diagnosis completed (awareness + sophistication)
- [ ] Target audience clearly defined
- [ ] Copywriter selected based on diagnosis
- [ ] Mental conversation mapped (@robert-collier)

## Copy Structure

- [ ] Headline grabs attention and promises benefit
- [ ] Lead matches awareness level
- [ ] Story or mechanism present
- [ ] Proof stacked (testimonials, data, demonstrations)
- [ ] Offer clearly presented
- [ ] CTA is specific and unmistakable
- [ ] Urgency is legitimate, not manufactured

## Quality Gates

- [ ] Hopkins Audit: >= 85/100
- [ ] Sugarman 30 Triggers: >= 80% coverage
- [ ] Read aloud test passed (sounds natural)
- [ ] No grammar/spelling errors
- [ ] Consistent voice throughout

## Final Check

- [ ] Copy matches the diagnosed awareness level
- [ ] Copy addresses market sophistication stage
- [ ] All claims are substantiated
- [ ] CTA leads to correct destination
