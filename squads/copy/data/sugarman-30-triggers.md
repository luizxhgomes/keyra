# Sugarman 30 Psychological Triggers

Source: Joe Sugarman — "Triggers" / "The Adweek Copywriting Handbook"

**IMPORTANT:** Sugarman is a TOOL, not a copywriter. These triggers are a validation checklist.

## The 30 Triggers

| # | Trigger | Description | Check |
|---|---------|-------------|-------|
| 1 | **Feeling of Involvement** | Reader feels part of the process | [ ] |
| 2 | **Honesty** | Transparent about flaws and limitations | [ ] |
| 3 | **Integrity** | Consistent with brand promises | [ ] |
| 4 | **Credibility** | Believable claims with proof | [ ] |
| 5 | **Value & Proof of Value** | Clear ROI or benefit demonstration | [ ] |
| 6 | **Justify the Purchase** | Logical reasons to buy (for post-rationalization) | [ ] |
| 7 | **Greed** | "Get more for less" appeal | [ ] |
| 8 | **Establish Authority** | Expert positioning | [ ] |
| 9 | **Satisfaction Conviction** | Guarantee or risk reversal | [ ] |
| 10 | **Nature of Product** | Product matches audience need | [ ] |
| 11 | **Current Fads** | Leverages current trends | [ ] |
| 12 | **Timing** | Right moment for the offer | [ ] |
| 13 | **Desire to Belong** | Community or group identity | [ ] |
| 14 | **Desire to Collect** | Part of a set or collection | [ ] |
| 15 | **Curiosity** | Information gaps that pull forward | [ ] |
| 16 | **Sense of Urgency** | Legitimate reason to act now | [ ] |
| 17 | **Fear** | What they'll lose by NOT acting | [ ] |
| 18 | **Instant Gratification** | Immediate benefit or access | [ ] |
| 19 | **Exclusivity** | Not available to everyone | [ ] |
| 20 | **Simplicity** | Easy to understand and act on | [ ] |
| 21 | **Human Relationships** | Personal connection and rapport | [ ] |
| 22 | **Guilt** | Reciprocity from free value given | [ ] |
| 23 | **Specificity** | Specific numbers and details | [ ] |
| 24 | **Familiarity** | Known concepts and references | [ ] |
| 25 | **Hope** | Future better self/situation | [ ] |
| 26 | **Pattern Interrupt** | Breaks expectations, grabs attention | [ ] |
| 27 | **Linking** | Connects to something they already value | [ ] |
| 28 | **Storytelling** | Narrative that carries the message | [ ] |
| 29 | **Mental Engagement** | Makes reader think and participate | [ ] |
| 30 | **Harmonize** | Resolves internal conflict about buying | [ ] |

## Scoring

- **Total triggers:** 30
- **Minimum coverage:** 80% (24/30)
- **Formula:** (triggers_present / 30) * 100

## Verdict

| Coverage | Verdict |
|----------|---------|
| >= 80% | APPROVED |
| 60-79% | NEEDS WORK — add missing triggers |
| < 60% | REJECT — fundamental persuasion gaps |
