# Research Quality Gate

Checklist de qualidade para validar o report final de deep research.

## Criteria

### Completeness (Must Pass)
- [ ] Todas as primary questions do prompt foram respondidas
- [ ] Executive summary presente e conciso (3-5 bullet points)
- [ ] Recommendations sao acionaveis e traceaveis aos findings

### Source Quality (Must Pass)
- [ ] Minimo 3 fontes independentes para findings de alta confianca
- [ ] Nenhum finding critico baseado em fonte unica
- [ ] Fontes classificadas por tier (Primary/Secondary/Tertiary)

### Accuracy
- [ ] Dados contraditorios documentados com ambas as perspectivas
- [ ] Confidence levels consistentes com evidencia apresentada
- [ ] Dados desatualizados (>12 meses) marcados como potentially stale

### Actionability
- [ ] Cada recomendacao tem pelo menos 2 findings de suporte
- [ ] Next steps claramente definidos
- [ ] Gaps e areas para pesquisa adicional documentados

### Structure
- [ ] Report segue template padrao do squad
- [ ] Tabelas e matrices formatadas corretamente
- [ ] Appendix com source registry completo

## Verdicts

| Verdict | Criteria | Action |
|---------|----------|--------|
| PASS | All Must Pass + >= 80% total | Deliver report |
| NEEDS_REVISION | Must Pass OK, < 80% total | Revise specific sections |
| FAIL | Any Must Pass failed | Re-execute research pipeline |
