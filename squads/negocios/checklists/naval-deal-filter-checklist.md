# Checklist: Filtro Naval de Deal

## Pre-Filtro Rapido (antes de investir tempo)

### Criterios de Entrada
- [ ] O deal tem asymmetric upside? (downside limitado, upside alto)
- [ ] O deal compoe no longo prazo? (nao e one-shot)
- [ ] Existe leverage real? (code, media, capital — nao so labor)
- [ ] Voce esta na posicao de principal? (equity, decisao, upside variavel)
- [ ] Usa seu specific knowledge? (moat incopiavel)
- [ ] As pessoas envolvidas jogam long-term games?
- [ ] O deal esta alinhado com quem voce realmente e? (autenticidade)

### Red Flags (qualquer SIM = reavaliar)
- [ ] Upside simetrico (ganho = perda potencial)
- [ ] Voce e 100% agente (fee fixo, zero equity)
- [ ] Depende 100% do seu tempo (zero leverage)
- [ ] Pessoas envolvidas tem historico de curto prazo
- [ ] Voce nao faria de graca (desalinhado)
- [ ] Nao compoe — resultado some quando para

### Veredicto
- [ ] **PLAY** (7+ criterios positivos, 0 red flags)
- [ ] **NEGOTIATE** (5-6 criterios positivos, red flags gerenciaveis)
- [ ] **PASS** (3-4 criterios positivos, red flags multiplos)
- [ ] **HARD NO** (< 3 criterios positivos — "if you can't decide, the answer is no")

### Rota Pos-Filtro
- [ ] PLAY → Encaminhar para diagnostico completo (Goldratt + Hormozi)
- [ ] NEGOTIATE → Redesenhar termos antes de prosseguir
- [ ] PASS → Calcular hourly rate, redirecionar tempo
- [ ] HARD NO → Sair imediatamente, registrar motivo
