# Checklist: Oferta irrecusável

> Validar a oferta ANTES de apresentar ao prospect. Hormozi + Goldratt + Cialdini.

## 1. Fundação: Mafia Offer (Goldratt)
- [ ] Oferta baseada em vantagem operacional real (não apenas desconto)
- [ ] Concorrência não consegue copiar facilmente
- [ ] Resolve o gargalo principal do cliente
- [ ] ROI claro e calculável para o cliente

## 2. Empacotamento: Grand Slam (Hormozi)
- [ ] **Oferta principal** clara e específica
- [ ] **Bônus 1** mata objeção de tempo ("muito demorado")
- [ ] **Bônus 2** mata objeção de esforço ("muito difícil")
- [ ] **Bônus 3** mata objeção de risco ("e se não funcionar?")
- [ ] **Garantia reversa** transfere risco para o vendedor
- [ ] **Naming** segue fórmula MAGIC (Make A Grand Inspiring Claim)
- [ ] **Pricing** posicionado por valor entregue, não custo

## 3. Value Equation
- [ ] Dream Outcome definido em linguagem do cliente
- [ ] Perceived Likelihood maximizada (provas, cases)
- [ ] Time Delay minimizado (entrega rápida ou resultado rápido)
- [ ] Effort & Sacrifice minimizado (facilidade de uso)
- [ ] Score V = (DO x PL) / (TD x ES) acima do baseline do mercado

## 4. Gatilhos embutidos (Cialdini)
- [ ] Reciprocidade: valor entregue antes do pedido
- [ ] Consistência: micro-yeses na jornada
- [ ] Prova Social: "outros como você já fizeram"
- [ ] Autoridade: credenciais visíveis
- [ ] Escassez: limitação REAL (vagas, capacidade, prazo)
- [ ] Unidade: "nós somos do mesmo tipo"

## 5. Validação final (Rockefeller)
- [ ] A oferta serve à visão de longo prazo (não só o deal)
- [ ] Margem protegida (não estamos cedendo demais)
- [ ] Oferta é sustentável em escala
- [ ] Os 7 mandamentos do Rockefeller respeitados

## Veredicto
- **18+ itens:** OFERTA PRONTA, apresentar com confiança
- **14 a 17 itens:** REVISAR, gaps específicos precisam de ajuste
- **Menos de 14 itens:** REDESENHAR, oferta incompleta, voltar para fase 2
