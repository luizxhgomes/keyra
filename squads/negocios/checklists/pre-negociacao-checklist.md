# Checklist: Pré-negociação

> Validar ANTES de entrar em qualquer negociação. Nenhuma conversa começa sem este gate.

## 1. Diagnóstico do sistema (Goldratt)
- [ ] Gargalo do cliente identificado (5 Focusing Steps aplicados)
- [ ] Throughput do cliente mapeado (T, I, OE)
- [ ] Cadeia de valor do cliente entendida

## 2. Oferta estruturada (Hormozi)
- [ ] Value Equation completa (Dream Outcome x Likelihood / Time x Effort)
- [ ] Oferta principal definida
- [ ] Bônus stack montado (cada bônus mata uma objeção)
- [ ] Garantia reversa definida (risco do vendedor)
- [ ] Naming MAGIC aplicado
- [ ] Pricing por valor, não por custo

## 3. Mesa preparada (Ury)
- [ ] Interesses reais mapeados (não posições)
- [ ] BATNA própria definida e forte
- [ ] BATNA do prospect estimada
- [ ] ZOPA calculada (zona de acordo possível)
- [ ] Critérios objetivos listados (3+ critérios)
- [ ] Agenda da reunião estruturada

## 4. Terreno preparado (Cialdini)
- [ ] Pre-Suasion aplicada (frame antes da mensagem)
- [ ] Prova social preparada (cases, depoimentos)
- [ ] Autoridade estabelecida (credenciais, resultados)
- [ ] Reciprocidade plantada (valor entregue antes de pedir)

## 5. Inteligência (Rockefeller)
- [ ] Informação sobre o prospect coletada
- [ ] Decision maker identificado
- [ ] Timeline do prospect conhecida
- [ ] Budget estimado ou confirmado
- [ ] Concorrentes na mesa identificados

## Veredicto
- **15+ itens marcados:** GO, prosseguir com confiança
- **10 a 14 itens:** GO COM CAUTELA, gaps identificados, mitigar durante a conversa
- **Menos de 10 itens:** NO-GO, voltar para fase de preparação
