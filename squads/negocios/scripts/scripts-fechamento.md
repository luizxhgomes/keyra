# Scripts de fechamento

> Dois caminhos: consultivo (Voss com Ackerman) e transacional (Belfort com Straight Line).

---

## Caminho A: Fechamento consultivo (Voss)

### Ackerman Bargaining: quando o prospect quer negociar preço

**Rodada 1 (65% do alvo):**
```
"Com base em tudo que discutimos, consigo fazer por R$ [65% do alvo].
Sei que é um investimento, mas considerando [resultado prometido],
o retorno é [X]x em [prazo]."
```

**Rodada 2 (85% do alvo), após objeção:**
```
"Olha, conversei internamente e consegui um ajuste.
Consigo R$ [85%], mas isso já está no limite do que posso fazer
mantendo [escopo completo]."
```

**Rodada 3 (95% do alvo), após segunda objeção:**
```
"Vou ser muito transparente: R$ [95%] é o máximo que consigo
sem cortar componentes. E esse valor só consigo manter até [data]."
```

**Final (100% com número não redondo):**
```
"R$ [valor preciso, ex: 14.738]. Esse é o número final.
Inclui tudo que discutimos, mais [bônus adicional pequeno].
Consigo manter isso se fecharmos até [data]."
```

**Por que número não redondo:** Parece calculado, não arbitrário. Gera mais confiança.

### Fechamento com "That's Right" (Voss)

**Antes de propor o fechamento, buscar o "That's Right":**

```
"Então, se eu entendi bem: o principal problema de vocês é [gargalo].
Vocês já tentaram [alternativas] mas não resolveu porque [motivo].
O que vocês realmente precisam é [resultado real, não o pedido inicial].
E o timing é importante porque [urgência real].

Estou entendendo certo?"
```

**Prospect responde "É, é isso mesmo" (That's Right), e aí é o momento de fechar:**

```
"Ótimo. Então a proposta é: [oferta resumida em 1 frase].
Começamos [data]. Faz sentido para vocês?"
```

## Caminho B: Fechamento transacional (Belfort)

### Straight Line Close: rápido e direto

**Checar Three Tens antes de fechar:**
- Produto: o prospect acredita que resolve? (precisa ser 8+)
- Vendedor: o prospect confia em mim? (precisa ser 8+)
- Empresa: o prospect confia na organização? (precisa ser 7+)

**Se Three Tens estão altos:**
```
"[Nome], tudo que discutimos faz sentido para você, certo?
[Pausa, esperar confirmação]

Perfeito. Então vamos fazer o seguinte: eu preparo [próximo passo]
e começamos [data]. Como você prefere o pagamento, [opção A] ou [opção B]?"
```

**Tonalidade:** Tom de fato consumado. Não perguntar SE vai fechar, perguntar COMO.

---

### Looping Close (Belfort): se ainda há hesitação

```
"[Nome], eu entendo. Deixa eu perguntar uma coisa:
em uma escala de 1 a 10, qual a sua confiança de que
[produto ou solução] resolve [problema]?"

[Se < 8: trabalhar essa certeza específica]
[Se >= 8: "O que falta para ser um 10?"]
[Resolver o gap, tentar fechar novamente]
```

## Fechamento alternativo: Structured Choice (Trump)

**Quando o prospect está indeciso entre opções:**

```
"Temos duas opções que fazem sentido para o cenário de vocês:

Opção A: [escopo completo] por R$ [valor maior]
Opção B: [escopo essencial] por R$ [valor menor]

Qual das duas se encaixa melhor no momento?"
```

**Por que funciona:** Ambas as opções são favoráveis. A pergunta não é "se", é "qual".

---

## Fechamento de urgência real (Cialdini)

**Somente quando a urgência é REAL (nunca fabricada):**

```
"Uma coisa importante: [restrição real: vagas, capacidade, deadline].
Não estou tentando pressionar, é uma questão operacional.
Se fizer sentido para vocês, o ideal é definirmos até [data].
Depois disso, [consequência real]."
```

---

## Pós-fechamento: confirmar e ancorar

**Imediatamente após o "sim":**

```
"Excelente, [Nome]. Você tomou uma ótima decisão.
Vou te enviar [documento ou contrato] em [prazo].
O próximo passo é [ação concreta + data].
Qualquer dúvida até lá, estou disponível."
```

**Por que funciona:** Consistência (Cialdini). Reforçar a decisão logo após o compromisso reduz o "buyer's remorse".
