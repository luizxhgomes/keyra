# Scripts de manejo de objeções

> Reframes prontos para as 10 objeções mais comuns. Voss + Belfort + Cohen.

---

## 1. "Está caro"

**Reframe (Hormozi):**
```
"Entendo. Deixa eu fazer uma pergunta: caro comparado a quê?

Se [seu problema] custa R$ [valor da dor] por mês e a solução
custa R$ [preço], em [prazo] você já recuperou o investimento
e tudo depois disso é lucro.

A pergunta não é se é caro. É: quanto está custando NÃO resolver?"
```

---

## 2. "Preciso pensar"

**Reframe (Voss):**
```
"Faz todo sentido. Parece que tem algo específico que está
te segurando. Posso perguntar o que seria?"
```

**Se não revelar (Belfort, via looping):**
```
"Normalmente quando alguém precisa pensar, é por um de três motivos:
preço, timing ou confiança na solução. Qual desses se parece mais
com o que você está sentindo?"
```

---

## 3. "Preciso falar com meu sócio ou chefe"

**Reframe (Voss):**
```
"Claro, faz sentido envolver [sócio ou chefe]. Me ajuda a entender:
o que você diria para ele sobre isso? Assim posso te dar
os pontos certos para a conversa."
```

**Alternativa (Belfort):**
```
"Perfeito. Que tal marcarmos uma conversa rápida com ele também?
Assim respondo qualquer dúvida direto e ninguém perde tempo
com telefone sem fio."
```

---

## 4. "Já tenho um fornecedor ou solução"

**Reframe (Cialdini + Voss):**
```
"Ótimo sinal. Significa que você já valoriza isso.

Me conta: se pudesse mudar UMA coisa na solução atual,
o que seria?"
```

**Por que funciona:** Consistência (já valoriza) + pergunta calibrada (expõe gap).

---

## 5. "Não é prioridade agora"

**Reframe (Goldratt):**
```
"Entendo que tem muita coisa acontecendo. Curiosidade:
qual é o gargalo principal de vocês hoje?

Pergunto porque, pela minha experiência, [problema que resolvemos]
geralmente é o gargalo invisível que trava tudo o mais."
```

---

## 6. "Me manda uma proposta por email"

**Reframe (Belfort):**
```
"Com certeza, vou preparar algo personalizado. Para garantir
que a proposta faça sentido para vocês, posso fazer mais
duas perguntas rápidas? Assim não mando algo genérico."
```

**Objetivo:** Manter a conversa, não perder para o email.

---

## 7. "O concorrente faz mais barato"

**Reframe (Hormozi):**
```
"Entendo. É uma comparação justa.

A diferença é que nós não vendemos [produto]. Vendemos [resultado].

O [concorrente] entrega [escopo deles]. Nós entregamos
[resultado concreto + garantia + bônus]. Quando você coloca
na ponta do lápis, o custo por resultado é menor com a gente."
```

---

## 8. "Deixa para o próximo trimestre"

**Reframe (Cohen + Cialdini):**
```
"Faz sentido planejar. Uma pergunta honesta: se começarmos
no próximo trimestre, vocês só vão ter resultado em [data futura].

Se começarmos agora, em [data próxima] vocês já estariam com
[resultado]. O custo de esperar 3 meses é R$ [valor da dor x 3]."
```

---

## 9. "Não tenho orçamento"

**Reframe (Ury):**
```
"Entendo a restrição de orçamento. Me ajuda com uma coisa:
se o orçamento não fosse limitação, isso seria a solução certa?

[Se sim:] Então vamos pensar juntos: existe uma forma de
começarmos menor e expandir conforme o resultado aparece?"
```

---

## 10. "Preciso ver mais resultados ou cases"

**Reframe (Cialdini):**
```
"Justo. Resultados falam mais que qualquer argumento.

Vou te mandar [case específico] de uma empresa do seu segmento
que estava na mesma situação. [Nome] da [empresa] pode inclusive
conversar com você diretamente se quiser."
```

**Por que funciona:** Prova social + autoridade + reciprocidade.

---

## Regra de ouro para objeções (Belfort)

> Cada objeção é um pedido por mais certeza.
> Não rebata. Não discuta. Não fique defensivo.
> Reconheça, faça o reframe, aumente a certeza e tente fechar novamente.
