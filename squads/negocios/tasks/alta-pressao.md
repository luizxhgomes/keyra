# Cenários de Alta Pressão

## Fase: 6 — Alta Pressão (Reserva Estratégica)
## Agentes Ativos: Rockefeller → Trump + Cohen

---

## AVISO
**Esta fase requer autorização explícita de Rockefeller.**
Só usar quando a alavancagem é REAL e o deal é PONTUAL.
NUNCA em mercados de recorrência.

## Objetivo
Extrair máximo valor em cenários onde você tem posição de força clara.
Trump força, Cohen calibra.

## Pré-Condições
- [ ] Alavancagem REAL confirmada (não percebida)
- [ ] Deal PONTUAL (não recorrente)
- [ ] BATNA forte (pode caminhar para longe)
- [ ] Rockefeller autorizou

## Steps

### Step 0: Gate de Autorização (Rockefeller)
**OBRIGATÓRIO — sem este gate, a execução é BLOQUEADA.**

Verificar:
1. **Rockefeller autorizou explicitamente?** → Se NÃO → BLOQUEAR. Retornar à fase anterior.
2. **Alavancagem é REAL?** → Confirmar com dados concretos (contrato, dependência, exclusividade), não percepção.
3. **Deal é PONTUAL?** → Se recorrente/relacionamento de longo prazo → BLOQUEAR. Usar Voss (fase 5) em vez disso.
4. **BATNA é forte?** → Você pode caminhar para longe sem perda material?

| Critério | Requisito | Status |
|----------|-----------|--------|
| Autorização Rockefeller | Explícita | [ ] |
| Alavancagem real | Dados concretos | [ ] |
| Deal pontual | Não recorrente | [ ] |
| BATNA forte | Walk-away viável | [ ] |

**Decisão:**
- 4/4 confirmados → **PROSSEGUIR** para Step 1
- Qualquer item falho → **BLOQUEAR** — escalar para Rockefeller com justificativa

### Step 1: Avaliação de Alavancagem (Cohen)
Analisar as 3 variáveis com scoring objetivo:

| Variável | Pergunta | Score (1-10) |
|----------|----------|-------------|
| **Poder** | Quem tem mais opções? Quem depende mais de quem? | [ ] |
| **Tempo** | Quem tem mais urgência? Deadline de quem? | [ ] |
| **Informação** | Quem sabe mais? Que informação falta? | [ ] |

**Thresholds de decisão:**
- Média >= 8 → **PROSSEGUIR** com confiança
- Média 6-7 → **PROSSEGUIR COM CAUTELA** — Cohen monitora de perto
- Média < 6 → **ABORTAR** — alavancagem insuficiente, retornar para fase 5 (Voss/Belfort)

### Step 2: Calibragem Emocional (Cohen)
Antes de executar:
- [ ] Você PRECISA desse deal? (Se sim, não use esta fase)
- [ ] Você pode caminhar para longe sem perda significativa?
- [ ] O relacionamento é descartável?
- [ ] "Care, but not that much" — seu desapego é genuíno?

### Step 3: Ancoragem Extrema (Trump)
1. **Door in the Face** — Comece com pedido absurdamente alto
2. **Silêncio** — Não recue imediatamente
3. **Structured choice** — Ofereça 2 opções (ambas favoráveis a você):
   - "Você prefere Opção A (R$X com escopo Y) ou Opção B (R$X' com escopo Y')?"
4. **Recuo calculado** — Reduza para o que você realmente quer
   - A redução parecerá uma concessão generosa

### Step 4: Imprevisibilidade Controlada (Trump)
- Mude o ritmo inesperadamente
- Adicione um pedido novo quando acharem que está quase fechado
- Projete confiança absoluta (nunca mostre dúvida)
- "Walk away" — ameace sair (só se puder realmente sair)

### Step 5: Controle de Danos (Cohen)
DURANTE a execução, Cohen monitora:
- [ ] A escalada está no nível certo?
- [ ] O relacionamento está danificado irremediavelmente?
- [ ] O outro lado está a ponto de desistir?
- [ ] É hora de recuar?

**Se qualquer sinal de ruptura:** Cohen intervém, desescala, salva o deal.

## Output
- Deal fechado em termos favoráveis OU
- Recuo estratégico com relacionamento preservado
- Lições aprendidas para próxima vez
