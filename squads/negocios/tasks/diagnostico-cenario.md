# Diagnóstico de Cenário de Negociação

## Fase: 1 — Diagnóstico do Sistema
## Agentes Ativos: Rockefeller → Naval + Goldratt + Hormozi

---

## Objetivo
Diagnosticar o cenário completo antes de qualquer ação. Identificar gargalos
do sistema comercial, mapear a equação de valor do cliente, e decidir se o
jogo vale a pena.

## Pré-Condições
- Descrição do cenário de negociação (produto/serviço, mercado, prospect)
- Contexto do deal (valor, urgência, recorrência vs pontual)

## Steps

### Step 0: Pre-Filtro Estrategico (Naval)
Antes de diagnosticar, avaliar se o jogo vale a pena:
1. **Asymmetric upside?** — O downside e limitado mas o upside e alto?
2. **Compounding?** — Esse deal cresce sozinho com o tempo?
3. **Leverage?** — Existe code, media ou capital envolvido? Ou e so labor?
4. **Principal?** — Estamos na posicao de dono ou de prestador?

**Se 3+ respostas = NAO → Considerar NO-GO antes de investir tempo no diagnostico.**
**Se 3+ respostas = SIM → Prosseguir para Step 1.**

### Step 1: Mapeamento do Sistema (Goldratt)
**Elicit:** Peça ao usuário para descrever:
1. O que você vende? (produto/serviço)
2. Para quem? (perfil do cliente ideal)
3. Qual o tamanho do deal? (valor aproximado)
4. Recorrente ou pontual?
5. Qual o principal problema que o cliente quer resolver?
6. Quem é a concorrência?

### Step 2: 5 Focusing Steps (Goldratt)
Aplicar os 5 Focusing Steps ao sistema comercial:
1. **IDENTIFY** — Qual é o gargalo? (geração de leads? qualificação? fechamento? retenção?)
2. **EXPLOIT** — Como extrair o máximo do gargalo sem investimento extra?
3. **SUBORDINATE** — O que mais precisa mudar para servir o gargalo?
4. **ELEVATE** — Que investimento quebraria o gargalo?
5. **REPEAT** — Qual seria o próximo gargalo após resolver este?

### Step 3: Equação de Valor (Hormozi)
Mapear a equação de valor do cliente:
- **Dream Outcome** — Qual o resultado ideal que o cliente quer?
- **Perceived Likelihood** — Quão provável o cliente acha que vai funcionar? (1-10)
- **Time Delay** — Quanto tempo até ver resultado? (menor = melhor)
- **Effort & Sacrifice** — Quanto esforço o cliente precisa investir? (menor = melhor)

**Fórmula:** Valor = (Dream Outcome x Likelihood) / (Time x Effort)

### Step 4: Decisão Estratégica (Rockefeller)
Com base no diagnóstico, decidir:
- [ ] O gargalo é claro e atacável?
- [ ] A equação de valor justifica o esforço?
- [ ] Temos vantagem competitiva real (não percebida)?
- [ ] O deal serve à visão de longo prazo?

**Veredicto:**
- **GO** — Prosseguir para Fase 2 (Construir Oferta)
- **REDESIGN** — O sistema precisa de ajuste antes de negociar
- **NO-GO** — O deal não vale a pena

## Output
- Relatório de diagnóstico com gargalo identificado
- Equação de valor mapeada
- Recomendação: GO / REDESIGN / NO-GO
- Próximo passo sugerido
