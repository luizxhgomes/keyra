# Gerar Playbook Prático

## Tipo: Ferramenta do Mandatário
## Agentes Ativos: Rockefeller orquestra os especialistas necessários

---

## Objetivo
Gerar um playbook prático e acionável para uma situação específica
de negociação, combinando frameworks dos agentes relevantes.

## Steps

### Step 1: Definir Situação
**Elicit:**
1. Tipo de negociação (venda, parceria, fornecedor, salário, contrato)
2. Contexto (B2B/B2C, ticket, recorrência, urgência)
3. Seu nível de alavancagem (alto/médio/baixo)
4. Objetivo específico (fechar, renegociar, expandir, defender)

### Step 2: Selecionar Agentes
Com base na situação, Rockefeller seleciona os agentes relevantes:

| Situação | Agentes Ativos |
|----------|---------------|
| Venda consultiva B2B | Goldratt → Hormozi → Ury → Voss |
| Venda transacional | Hormozi → Belfort + Cialdini |
| Renegociação de contrato | Ury + Cohen → Voss |
| Fornecedor (você compra) | Cohen + Ury → (Trump se alavancagem) |
| Negociação de salário | Ury + Cohen → Voss |
| Parceria estratégica | Goldratt + Ury → Cialdini |
| Deal de alta pressão | Trump + Cohen (com autorização) |

### Step 3: Gerar Playbook
Estrutura do playbook:
1. **Contexto** — Resumo da situação
2. **Diagnóstico** — Gargalo e equação de valor
3. **Preparação** — BATNA, interesses, critérios
4. **Oferta** — Estrutura e gatilhos
5. **Script de Abertura** — Primeiras palavras
6. **Manejo de Objeções** — Top 3 objeções + respostas
7. **Fechamento** — Estratégia + Ackerman/Straight Line
8. **Follow-up** — Cadência pós-conversa
9. **Plano B** — Se não fechar, próximo passo

### Step 4: Validação
Checar contra princípios de cada agente ativo.

## Output
- Playbook completo em formato acionável
- Scripts prontos para uso
- Checklist pré-negociação
- Manejo de objeções mapeado
